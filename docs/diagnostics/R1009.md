# R1009 — invalid metadata

**Message patterns:** `Metadata must be Symbol,Keyword,String,Map or Vector` · `Metadata can only be applied to IMetas`

## Cause

Either the metadata form after `^` is of an unsupported type, or the
target form cannot carry metadata. `^meta` accepts a map (used as-is),
a keyword (`^:kw` → `{:kw true}`), a symbol or string (`^Tag` →
`{:tag Tag}`), or a vector (`^[...]` → `{:param-tags [...]}`) — and
the following form must support metadata (symbols and collections do;
numbers, strings, keywords, and nil do not).

## Example

```clojure
^42 foo        ; Metadata must be Symbol,Keyword,String,Map or Vector
^:private 42   ; Metadata can only be applied to IMetas
```

## Fix

Use a supported metadata shape, and attach it to a form that can hold
it (a symbol, list, vector, map, or set). To associate data with a
plain value, use a map entry or `with-meta` on a collection wrapping
it instead.
