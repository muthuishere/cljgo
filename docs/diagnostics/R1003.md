# R1003 — map literal with odd number of forms

**Message pattern:** `Map literal must contain an even number of forms`

## Cause

A map literal `{...}` holds key/value pairs, so it must contain an
even number of forms. An odd count means a key has no value (or a
value has no key) — often a forgotten value, or a stray form pasted
into the map.

## Example

```clojure
{:a 1 :b}
```

`:b` has no value.

## Fix

Add the missing value (e.g. `{:a 1 :b nil}`), remove the dangling key,
or check for a misplaced delimiter that cut a pair in half. The
diagnostic points at the `{` that opens the literal.
